React

React is a JavaScript library for building user interfaces.

It is notable for its use in crafting single-page apps and creating reusable UI components through the use of JSX.

JSX allows you to write HTML within JavaScript. It converts HTML tags into react elements.

JSX Syntax

Hereâ€™s a simple example of JSX.

const element = <h1>Hello, world!</h1>;

You can embed JavaScript expressions inside JSX by wrapping them in curly braces. The Example below will render "Hello, React Developer!" on the page.

const name = 'React Developer';
const element = <h1>Hello, {name}!</h1>;

After compilation, JSX expressions become regular JavaScript objects. This means you can use JSX inside if statements and for loops, assign it to variables, accept it as arguments, and return it from functions:

function getGreeting(user) {
  if (user) {
    return <h1>Hello, {user}!</h1>;
  }
  return <h1>Hello, Stranger.</h1>;
}

JSX Attributes

You can pass attributes to JSX elements just like you do in HTML:

const element = <img src="logo.png" alt="Logo" />;

Note that in JSX, you use `camelCase` for naming attributes instead of `kebab-case`. For example, `class` in HTML becomes `className` in JSX:

const element = <div className="container"></div>;

JSX Children

If a tag is empty, you can close it immediately with `/>`, like XML:

const element = <img src="logo.png" alt="Logo" />;

You can also nest elements inside other elements:

const element = (
  <div>
    <h1>Hello, World!</h1>
    <p>Welcome to learning React.</p>
  </div>
);

JSX and JavaScript Functions

JSX can represent JavaScript functions as well:

function formatName(user) {
  return user.firstName + ' ' + user.lastName;
}

const user = {
  firstName: 'Harper',
  lastName: 'Perez'
};

const element = <h1>Hello, {formatName(user)}!</h1>;

In this example, the `formatName` function is used inside JSX to generate a greeting.

Creating a React Application

Ensure you have Node.js installed. Node.js comes with npm (Node Package Manager), which is used to install React.

To create your React application, open your terminal in Visual Studio Code; enter and run the following command.

Note that you can replace my-react-app to any name you want to use for the project's folder.

npx create-react-app my-react-app

After this has been run, you will want to navigate to your project directory.

cd my-react-app

Now that the project is set up, you can start the sever with the command below.

npm start
