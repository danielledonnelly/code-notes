React

React is a JavaScript library for building user interface.

It is notable for its use in crafting single-page apps and creating reusable UI components.

Getting Started

Before you get bstarted, ensure you have Node.js installed. Node.js comes with npm (Node Package Manager), which is used to install React.

To create your React application, open your terminal in Visual Studio Code; enter and run the following command.

Note that you can replace my-react-app to any name you want to use for the project's folder.

npx create-react-app my-react-app

After this has been run, you will want to navigate to your project directory.

cd my-react-app

Now that the project is set up, you can start the sever with the command below.

npm start
